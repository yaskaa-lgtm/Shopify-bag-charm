## D8 — QA GATE (not counted in build 10h; ~1.0h)
Run **every** checklist line in G §3 (functional, mobile/390px, compliance, email, tracking). Any fail = fix before D9. Do not open ad taps with an open fail on F1–F10, C1–C6, or P1–P8.

## D9 — Launch
Turn on TikTok campaigns. Watch CPA vs **break-even $22.5** (target < $18). Keep the 15/11/2026 paid-cut date in view — front-load learning, don't scale on unproven creatives.

---

## Critical path (the blockers that gate everything)
`CJ sample/specs → product created → images → PDP built → Vitals offers → compliance live → tracking verified → QA gate → launch.`
If any single blocker slips, launch slips. The most common slip is **CJ sample/specs (D1)** — start it the moment you begin.
